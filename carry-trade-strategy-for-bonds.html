Trading rules
Bond selection from 30-country universe

Top 33% high-yield countries identified
Equal-weighted portfolio allocation
Monthly portfolio rebalancing
Why it works
The bond carry strategy generates profit by targeting high-yield bonds associated with non-diversifiable risk factors, such as political unrest, unstable property rights, or continuous high inflation. These high yields incorporate a risk premium, which contributes to the strategyâ€™s profitability.
Hedging stocks in bear markets?
The strategy selects countries based on bond yields, favoring those with higher yields. However, these countries also have a higher risk of default, particularly during unfavorable global economic conditions. As a result, this approach is not an ideal hedge for equity market risk.

Here is the Python code for the trading rules using Backtrader:
import backtrader as bt class BondSelectionStrategy(bt.Strategy): def __init__(self): self.bond_universe = self.datas def next(self): # Sort bonds by yield (descending) sorted_bonds = sorted(self.bond_universe, key=lambda bond: bond.close[0], reverse=True) # Select the top 33% high-yield countries top_n = int(len(sorted_bonds) * 0.33) selected_bonds = sorted_bonds[:top_n] # Equal-weighted portfolio allocation weight = 1.0 / len(selected_bonds) # Rebalance the portfolio monthly for bond in self.bond_universe: if bond in selected_bonds: self.order_target_percent(bond, target=weight) else: self.order_target_percent(bond, target=0.0) if __name__ == '__main__': cerebro = bt.Cerebro() # Add data feeds to cerebro for the 30-country bond universe # cerebro.adddata(...) cerebro.addstrategy(BondSelectionStrategy) cerebro.run()
